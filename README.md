# QuantScope

Hardware-aware neural-network quantization, sensitivity analysis, and numerical debugging for edge accelerators.

## Status

QuantScope is under active development.

## Planned capabilities

- Post-training quantization
- Quantization-aware training
- Custom calibration observers
- Layer-level numerical-error analysis
- Sensitivity analysis
- Hardware-aware mixed-precision search
- Numerical-regression testing
- Reproducible experiment reports

## Disclaimer

QuantScope does not use, emulate, or claim compatibility with proprietary Quadric technology.

The included hardware profile is fictional and illustrative. CPU latency must not be presented as NPU latency. Simulated INT4 arithmetic and analytical hardware costs must be labeled clearly.
